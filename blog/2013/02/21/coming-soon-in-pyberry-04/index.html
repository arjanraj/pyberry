Coming Soon in Pyberry 0.4

Pyberry version 0.3 has been tagged for release but not packaged; you can get a copy of the source by doing a
git clone git://git.code.sf.net/p/pyberry/code pyberry-code

I’d try to package it but as of now I don’t know anybody following development/interested in a newer version; feel free to leave a comment if you are and I’ll take some time to package it.

I’ve instead been working on release 0.4, which will include the ability to add books by looking by something besides ISBN.  I’m currently implementing add by title functionality but once that’s implemented more methods should follow easily and quickly.

No timeframe because things come up and I know as soon as I give a timeframe I’ll miss it.  Stay tuned for more updates.
