Major Release: 1.0

This is it, I’ve done it. I’ve tagged and pushed release 1.0. I’ve done as much testing as I can figure, but there’s surely still bugs. If you find a bug, please report it on GitHub so I can fix it!

I also discovered the GitHub Releases feature, so I’ll be phasing out the “Downloads” page of this website soon.

Download Pyberry 1.0 here.

I’m very excited for this huge milestone for me (it’s my first project to hit 1.0/be reasonably stable), so I hope it benefits somebody else. I’m always open to feature requests or people looking to help. You can find various ways to contact me at my website, or just fork or open an issue on GitHub.

Thank you!
